# About DueTo

DueTo is a task management and organization tool based on the similar website Todoist.com. It allows users create tasks, seperated by different projects, and organize them according to user preference using drag-and-drop functionallity along with a selected "Board" or "List" layout.

[Live Site](https://todo-clone.onrender.com) | [Wiki](https://github.com/EricSapsford/ToDo/wiki)
